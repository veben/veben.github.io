## Local Developement

1. Clone this repository
    ```sh
    git clone https://github.com/veben/veben.github.io.git
    ```

2. Install Ruby if not alreday done
3. Install Github pages
    ```sh
    gem install github-pages
    ```
    
    > "This mirrors the plug-ins used by GitHub Pages on your local machine including Jekyll, Sass, etc."

4. Serve the site and watch for markup/sass changes
    ```sh
    jekyll serve
    ```
5. View your website at: http://127.0.0.1:4000/
More information: https://www.smashingmagazine.com/2014/08/build-blog-jekyll-github-pages/
